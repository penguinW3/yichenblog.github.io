本文提供了可用于 Hugo 内容文件的基本 Markdown 语法示例，还展示了基本 HTML 元素是否在 Hugo 主题中使用 CSS 装饰

1、标题

以下 HTML<h1>—<h6> 元素表示六级节标题。<h1>是最高的部分级别，而 <h6> 是最低的。

1语法：# 一级标题
2     ## 二级标题
3     ### 三级标题
4     #### 四级标题
5     ##### 五级标题
6     ###### 六级标题

2、字体

1语法： *倾斜*
2      **加粗**
3      ***倾斜加粗***
4      ~~删除线~~
5      ==高亮== 或 <mark>高亮</mark>
6      ^上标^ 或 <sup>上标</sup>
7      ~下标~ 或 <sub>下标</sub>

示例：

  • 倾斜
  • 加粗
  • 倾斜加粗
  • 删除线
  • 高亮
  • 上标
  • 下标

3、列表

无序

1语法：’+‘+空格+内容

示例：

  • 一级
    • 二级
      • 三级
        • 四级
          • …..
  • A
  • B
  • C

有序

1语法：1+.+内容

示例：

  1. 序号1
  2. 序号2
  3. ……

任务列表

1语法：空格+-空格+[空格] 任务名称 或 - [x] 任务名称

示例：

  • 任务名称

4、表格

1语法：
2    |表头A|表头B|表头C|表头D|
3    |--|--|--|--|
4    |`C`|`Java`|`python`|`Go`|

示例：

表头A 表头B 表头C 表头D
C Java Python Go

5、引用

1语法：>+空格+引用内容

示例：

  • 引用1

  • 引用2

  • 引用3

6、分割线

1语法：第一种：---  / 第二种：***

示例：

  • 第一种：—
  • 第二种：***

示例：

  • python
